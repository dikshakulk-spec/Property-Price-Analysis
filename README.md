Project Title

Stock Price Trend Analysis using SQL

📖 Project Overview

This project focuses on analyzing stock market data to identify market trends, investor behavior, sector performance, stock volatility, and trading patterns. Using SQL queries and data analysis techniques, the project extracts meaningful insights from large-scale stock market datasets.

The analysis helps businesses, investors, and stakeholders understand:

Which sectors perform best
How stock volatility affects investment decisions
Which age groups contribute most to trading activity
Investment opportunities across sectors
🎯 Objectives
Analyze sector-wise market performance
Identify highly volatile stocks
Study trader demographics and behavior
Explore investment opportunities
Generate business insights using SQL
🗂️ Dataset Information

The project uses four relational datasets:

Dataset	Description
Company Data	Company name, sector, market cap
Stock Price Data	Daily stock performance
Trade Data	Trade transactions
Trader Data	Trader demographics and behavior
Dataset Size
Company Data → 500 rows
Stock Price Data → 50,000 rows
Trade Data → 200,000 rows
Trader Data → 10,000 rows
